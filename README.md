# simple-interest-calculator

## 📌 Description
The Simple Interest Calculator is a basic program that calculates the simple interest based on the principal amount, rate of interest, and time period. It is designed to help users quickly understand how simple interest is computed using a standard mathematical formula.

---

## 🧮 Formula Used
The simple interest is calculated using the formula:

Simple Interest = (Principal × Rate × Time) / 100

Where:
- **Principal (P)** is the initial amount of money
- **Rate (R)** is the annual interest rate (in percentage)
- **Time (T)** is the time period in years

---

## ▶️ How It Works
1. The user provides:
   - Principal amount  
   - Rate of interest  
   - Time period  
2. The calculator applies the simple interest formula.
3. The calculated interest is displayed as output.

---

## 📝 Example

### Input:
- Principal = 1000  
- Rate = 5  
- Time = 2  

### Output:
- Simple Interest = 100  

---

## 🎯 Use Case
This calculator is useful for:
- Learning basic financial calculations
- Educational demonstrations
- Quick interest estimation for short-term investments
